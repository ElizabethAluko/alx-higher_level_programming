===========================
Test for 1-my_list.py file
===========================

1-my_list.py contains a class MyList that inherit from list.

MyList has a public instance method: print_sorted(self).

print_sorted(self) method prints a sorted (ascending sort) list of the list.

All the elements of the list will be of type int.

Usage
=====
    Import MyList class:
    >>> MyList = __import__("1-my-list").MyList

case 1:
    >>> new_list = MyList()
    >>> new_list.append(1)
    >>> new_list.append(5)
    >>> new_list.append(3)
    >>> new_list.append(9)
    >>> new_list.append(10)
    >>> new_list.print_sorted()
    [1, 3, 5, 9, 10]
    >>>

case 2:
    >>> new_list1 = MyList()
    >>> new_list1.print_sorted()
    []
